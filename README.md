# 🎬 Movie API

API REST desenvolvida com Spring Boot para gerenciamento de filmes.

## Tecnologias
- Java 19
- Spring Boot
- Spring Data JPA
- PostgreSQL
- Maven

## Funcionalidades
- ✅ Listar filmes
- ✅ Buscar filme por ID
- ✅ Cadastrar filme
- ✅ Atualizar filme
- ✅ Remover filme

## Endpoints

GET /filmes

GET /filmes/{id}

POST /filmes

PUT /filmes/{id}

DELETE /filmes/{id}
